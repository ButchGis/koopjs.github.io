Koop Sandbox

The sandbox allows you to develop a Koop Provider in the browser. Just specify your model, which is responsible for fetching data and translating it to GeoJSON. Then specify your URL pattern. When you press run (at the bottom of the code), a server endpoint will be available on RunKit infrastructure which will allow you to add data to the map on the second tab.

The example below shows the Koop Provider for the Portland TriMet bus service. It displays the location of all buses in the Portland, OR metropolitan area.
