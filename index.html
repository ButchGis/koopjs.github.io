Core Concepts

Koop is, first and foremost, a server. It is written in JavaScript and runs in the Node.js runtime environment. It acts as middleware for the Express web framework. It uses provider node modules to extract data from third party providers, transform that data into GeoJSON and load it into the cache, then serve it on the web in various formats.

Caches

Koop uses a cache to store data from third party providers. This helps reduce requests to other servers, bypass rate limiting issues, and speed up response times.
